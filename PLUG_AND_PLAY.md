# PLUG_AND_PLAY — rusty-series

> *Integration guide.*


## Feature Flags

| Feature | Default | Description |
|---------|---------|-------------|
| `std` | yes | Standard library |
| `alloc` | yes | Allocator support |

## Integration

Import `rusty_series` in your project to access the functionality.

```
use rusty_series::{};
```

## Compatibility

- **Rust edition**: 2021+
- **Targets**: All tier-1 Rust targets
